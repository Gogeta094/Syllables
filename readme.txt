Програма містить 3 вкладки: Main, Test і Letters. А також налаштування.
Для налаштування бази голосних, приголосних букв а також символів, які додатково потрібно вилучити, чи не вилучати заходимо в 
File -> Characters.
Також існує пункт меню File -> Settings. Тут ми можемо налаштувати:
 - Separator - символ, яким будуть розбиватися склади
 - Separation Mode - c-cc та cc-c. Це режими поділу на склади, у випадку, якщо в нас є 3 або більше приголосних і з них не було знайдено жодної
 висхідної послідовності. В 1 випадку ми ділимо після 1 приголосної. В 2 випадку - перед останньою
 - Characters Table - вибір таблиці сил приголосних.
 - Language - мова, для якої здійснюватимуться фонетичні перетворення
 - Phonetics Mode - прапорець для ввімкнення/ввимкнення режиму фонетики
 - Absolute Measures - прапорець для перемикання абсолютних/відносних величин в результуючій статистиці
 
Test - вкладка, призначена для моментального тестування складоподілу та фонетичних перетворень певних речень,слів, шматків тексту.

Main:
В даній вкладці, перед натисканням кнопки процес, потрібно вказати на теку, в якій міститься база текстів, які ми будемо аналізувати. Зробити це можна в меню File -> Open. 
Увага! Файли повинен бути у кодуванні UTF-8.
Після завантаження файлів - тиснемо кнопку "Process". Усі завантажені файли почнуть оброблятися в паралелі.
У випадку, якщо в процесі обробки певного слова, виникне помилка (наприклад знайдеться невідомий раніше символ) - це слово буде пропущено, а інформація про помилку в певному слові виведеться в текстове поле Errors log.
Прогрес обробки файлів відображатиметься в прогресбарах, які містяться напроти кожного тексту в лівій панелі.

Після обробки усіх файлів - результати можуть бути збережені на диску. Для цього потрібно зайти в File -> Save та вказати теку, в яку зберегти результат.
Результат міститиме 4 теки і 1 .csv файл: 
 - Syllables - тека, зі всіма текстами, розбитими на склади
 - FirstSyllables - тека, зі всіма текстами, розбитими на склади (взято лише перші склади)
 - SyllablesCVV - тека зі всіма текстами, розбитими на склади, що перетворенні в формат CV (C - Consonant, V - Vowel)
 - SyllablesFirstCVV.txt - тека зі всіма текстами, розбитими на склади (взято лише перші склади), що перетворенні в формат CV (C - Consonant, V - Vowel)
 - Statistics.csv - файл вихідної статистики для бази текстів
 
 Letters:
 Вкладка, в якій по аналогії з вкладкою Main можна завантажити базу текстів і згенерувати статистику букв.
 Статистика букв не є прив'язаною до певної мови, тому текст може будь-якої мови.
 Дії для завантаження файлів в пам'ять і експорту результату на диск є аналогічними, як і у вкладці Main, але в даному випадку генерується лише .csv файл зі статистикою.
